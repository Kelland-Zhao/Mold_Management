## V20260630.1_修复上模下模弹窗机台号选择框失效

- **问题**：上模/下模弹窗中机台号 Select2 下拉选择框无法正常使用
- **根因**：Select2 默认将下拉菜单渲染到 `<body>`，在 Bootstrap modal 中会被遮挡且焦点被 modal 陷阱拦截
- **修复**：以下 4 个 Script 文件的 `dynamicGenerationSelectOption` 函数中：
  1. Select2 初始化时添加 `dropdownParent: secTemp.closest('.modal')`，将下拉菜单挂载到对应 modal 容器内
  2. 绑定 `shown.bs.modal` 事件，modal 显示时刷新 Select2 宽高和定位（解决隐藏元素初始化导致的位置计算错误）
  - `Maintenance_Script.html`
  - `ChangeOver_Script.html`
  - `Test_Script.html`
  - `Repair_Script.html`
- **补充修复**：Select2 选择框在 modal 中宽度塌缩、选中内容溢出换行
  - **根因**：`select2-bootstrap-theme v0.1.0` 为 Bootstrap 3 设计（`display:table`），与项目实际使用的 Bootstrap 5.3.1 的 flexbox input-group 不兼容
  - **修复**：在 `CSS.html`（4 页共用）添加 `.input-group > .select2-container--bootstrap` 的 flex 样式覆盖，使 Select2 容器在 flex 布局下正常撑开宽度
